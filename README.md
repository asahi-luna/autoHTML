# 使用说明
> 功能：可以使用一个数组自动生成一个布局页面
## 1. v,h分别表示垂直方向（vertical）和水平方向（horizon）
### 1.1. 垂直方向（vertical）
```
    //autohtml.opt.js
    
    module.exports = [
        ['v,r1,200px'],
        ['v,r2,100px'],
        ['v,r3,300px']
    ];
```
